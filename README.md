## 📌 Project Overview
This project presents an interactive sales dashboard built to analyze business performance for a multi-category retail store.  
The goal is to transform raw data into actionable insights that support better decision-making and increase sales.

---

## 🎯 Objectives
- Identify peak sales periods (monthly & weekly)
- Analyze customer purchasing behavior
- Evaluate performance across categories, regions, and products
- Highlight top-performing and underperforming items
- Support data-driven business decisions

---

## 🛠️ Tools & Technologies
- Microsoft Excel  
- Power Query (Data Cleaning & Transformation)  
- Pivot Tables & Pivot Charts  
- Interactive Dashboard Design  

---

## 📊 Key Features
- Dynamic KPIs (Total Sales, Units Sold, Average Sales)  
- Year-over-Year comparison (2023 vs 2024)  
- Monthly & Weekly sales trends  
- Sales breakdown by category and region  
- Customer segmentation (Age & Gender)  
- Top & underperforming products analysis  
- Interactive filters (Category & Region slicers)  

---

## 🔍 Key Insights
- Weekdays generate higher sales than weekends  
- Sports and Home Appliances are top-performing categories  
- Regional performance varies, with some regions outperforming others  
- Customer segments show different buying behaviors
